# ColorPicker (개인용)

## 프로젝트 개요
'어몽어스' 스타일의 모바일 게임 프로젝트로, 상대의 색상을 추리하고 찾아내는 게임입니다.

**개발 기간**: 2024.02 ~ 2024.11 (약 9개월)  
**팀 구성**: 컴퓨터공학과 4명  

## 🔗 프로젝트 링크
- 🎮 **협업 Repository**: [ColorPicker Main](https://github.com/MinSeok0406/G1)
- 🛠️ **맵 에디터**: [2D Map Structure Creator](https://ldj5098.github.io/2D_map_structure_creator/)

## 프로젝트 특징
Unity2D 기반의 모바일 멀티플레이어 게임으로, 미니게임과 추리 요소를 결합한 소셜 게임입니다.

## 담당 역할

### 개발 지원 팀 (서브 리더)
- **프로세스 개선**: 개발 워크플로우 최적화
- **협업 도구 개발**: 팀 생산성 향상을 위한 도구 제작
- **UI/UX 및 디자인**: 게임 인터페이스 디자인
- **프로젝트 관리**: 일정 관리 및 산출물 검수
- **팀 간 가교 역할**: 메인 로직 팀과 개발 지원 팀 연결

### 주요 기여
- 산출물 검수 및 피드백
- 메인 로직 팀과의 협업 조율

## 주요 해결 과제

### 맵 설계기 개발
**문제 상황**
- 맵 구조 설계 단계에서 구두 설명으로는 소통이 어려움
- 오차 발생 및 프로젝트 지연

**해결 방안**
- 웹 기반 맵 에디터 자발적 1인 개발
- 팀원들이 직접 타일을 찍어 맵을 그릴 수 있는 툴 제공

**추가 개선**
- URL 쿼리스트링 방식으로 맵 데이터 공유 기능 추가
- 즉각적인 수정 및 공유 가능
